# Welcome to the library

This is the "book" side of RF Academy: an **interconnected** knowledge base on SDR and the HackRF, written to take you from "what is a wave?" all the way to "I detected a drone". Notes link to each other (click the `highlighted` links) and are **welded to the missions**: every concept has a button to go see it **live**.

**Where to start?**

1. The basics: [[ondes-radio]] → [[decibels]] → [[bruit-et-snr]]
2. Going digital: [[echantillonnage]] → [[iq]]
3. Seeing the signal: [[fft-spectre]] → [[waterfall]]
4. The hardware: [[sdr-architecture]] → [[hackrf]] → [[antennes]]
5. Understanding signals: [[modulations]]
6. In the field: [[workflow-live]] → [[bandes-a-explorer]] → [[decoder-vs-detecter]] → [[legal-securite]]

**Preparing for the amateur-radio exam?** A dedicated track covers the whole syllabus (regulations, technical, propagation, operating): start with [[examen-radioamateur]].

**How does it work?** Plug in your SDR (or stay on the simulator), open a mission, and come back to read the theory whenever a word escapes you. Practice and reading reinforce each other.
