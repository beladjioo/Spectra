# Électricité : les bases de l'examen

Le socle technique. Trois grandeurs : la **tension** U (volts), le **courant** I (ampères), la **résistance** R (ohms).

**Loi d'Ohm** : `U = R · I`. À mémoriser dans ses trois formes (`I = U/R`, `R = U/I`).

**Puissance** : `P = U · I` (watts), qui se réécrit `P = R·I²` ou `P = U²/R`. La puissance se relie aux [[decibels|décibels]] quand on compare deux niveaux.

**Associations de résistances** :
- En **série**, les résistances s'additionnent : `R = R₁ + R₂ + …`
- En **parallèle**, ce sont les inverses qui s'additionnent : `1/R = 1/R₁ + 1/R₂ + …` (le résultat est toujours plus petit que la plus petite).

**Courant continu / alternatif** : en continu (DC) la tension est constante ; en alternatif (AC) elle oscille (la radio, c'est de l'AC à haute fréquence). Pour un signal sinusoïdal, on distingue la valeur **crête** et la valeur **efficace** (RMS) : `U_eff = U_crête / √2`.

**Unités et préfixes** à manier sans hésiter : pico (p, 10⁻¹²), nano (n, 10⁻⁹), micro (µ, 10⁻⁶), milli (m, 10⁻³), kilo (k, 10³), méga (M, 10⁶), giga (G, 10⁹).
