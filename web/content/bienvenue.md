# Bienvenue dans la bibliothèque

Ceci est la partie « livre » de RF Academy : une base de connaissances **interconnectée** sur le SDR et le HackRF, écrite pour aller du « c'est quoi une onde ? » jusqu'au « j'ai détecté un drone ». Les notes sont reliées entre elles (clique les liens `surlignés`) et **soudées aux missions** : à chaque concept, un bouton pour aller le voir **en direct**.

**Par où commencer ?**

1. Les bases : [[ondes-radio]] → [[decibels]] → [[bruit-et-snr]]
2. Le numérique : [[echantillonnage]] → [[iq]]
3. Voir le signal : [[fft-spectre]] → [[waterfall]]
4. Le matériel : [[sdr-architecture]] → [[hackrf]] → [[antennes]]
5. Comprendre les signaux : [[modulations]]
6. Sur le terrain : [[workflow-live]] → [[bandes-a-explorer]] → [[decoder-vs-detecter]] → [[legal-securite]]

**Tu prépares l'examen radioamateur ?** Un parcours dédié couvre tout le programme (réglementation, technique, propagation, trafic) : commence par [[examen-radioamateur]].

**Comment ça marche ?** Branche ton HackRF (ou reste en simulateur), ouvre une mission, et reviens lire la théorie quand un mot t'échappe. La pratique et la lecture se renforcent.
